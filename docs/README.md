# MagicMakeup GitHub Pages Site

This folder is a static GitHub Pages site.

- `index.html`: MagicMakeup project page, served as the root of the project site.
- `magicmakeup/index.html`: legacy/local duplicate of the project page.
- `assets/css/site.css`: shared styles.
- `assets/js/site.js`: result interactions and comparison viewer.
- `assets/images/`: exported web images from the paper figures.

Target URL:

```text
https://wzy-cad.github.io/MagicMakeup/
```

## Publish

Create a GitHub repository named `MagicMakeup` under the `wzy-cad` account, then push this folder/repository to GitHub.

In that repository's settings:

1. Go to **Settings -> Pages**.
2. Set **Source** to the main branch.
3. Set **Folder** to `/docs`.
4. Save and wait for GitHub Pages to deploy.

If the account name or repository name changes, the URL changes as:

```text
https://<github-username>.github.io/<repository-name>/
```
